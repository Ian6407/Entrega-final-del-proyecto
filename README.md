## Descripción

Proyecto analítico bajo la metodología CRISP-DM que busca identificar la relación 
entre el acceso a internet y los resultados de las pruebas Saber 11 en los municipios 
de Bello, Itagüí y Envigado (Antioquia), usando Apache Spark como herramienta 
principal de procesamiento.

El detalle completo del análisis, hallazgos y metodología se encuentra en los 
documentos de cada entrega dentro de la carpeta `docs/`.
